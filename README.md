# task-3-dhillipprasanna
🔐 Password Generator in Python  A simple and beginner-friendly Python project that generates random passwords based on the length specified by the user. This project demonstrates the use of Python’s random and string modules to create secure passwords containing letters and numbers.

📌 Features

* Generate random passwords instantly
* User-defined password length
* Includes uppercase and lowercase letters
* Includes numeric digits
* Easy to understand and customize

🛠️ Technologies Used

* Python 3
* Random Module
* String Module

🚀 How It Works

1. The user enters the desired password length.
2. The program creates a pool of characters using letters and digits.
3. Random characters are selected from the pool.
4. A secure password of the specified length is generated and displayed.
